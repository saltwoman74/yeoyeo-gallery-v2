# 📖 여여부동산 AI 챗봇 사용 설명서

박혜경 소장님, 환영합니다! 이 챗봇은 소장님을 대신해 고객들에게 **매물 추천, 대출/세금 안내, 이미지 비교**를 해주는 똑똑한 비서입니다.

---

## 🚀 1. 실행하는 법 (아주 간단해요)
1.  **폴더 열기**: 바탕화면 등의 `791cf3f1-...` 폴더를 엽니다.
2.  **`index` 클릭**: `index` (또는 `index.html`) 파일을 더블클릭합니다.
3.  **끝!**: 인터넷 창이 열리면서 바로 상담 준비가 완료됩니다.
    *   *초록색 글씨로 "매물 ~개 로드 성공!"이 뜨면 정상입니다.*

---

## 🏠 2. 매물 올리는 법 (구글 엑셀)
매물은 소장님이 쓰시는 **구글 스프레드시트**에서 관리합니다. 내용은 자동으로 챗봇에 반영됩니다.

1.  **시트 열기**: [소장님의 구글 시트 바로가기](https://docs.google.com/spreadsheets/d/1Ajn0VVRqQfpjEimzmW7yorf7ecL9RKpXWpsCNj2QhsE/edit)
2.  **입력 하기**:
    *   **단지명, 동, 호수**: 정확히 적어주세요.
    *   **가격**: '8억 5천' 처럼 한글로 적어도 되고, '85000' 숫자로 적어도 됩니다.
    *   **특징**: "전망 굿", "시스템 에어컨" 등 자세히 적을수록 AI가 설명을 잘합니다.
    *   **이미지URL**: (선택사항) 사진 주소를 넣으면 챗봇이 사진도 보여줍니다.
3.  **저장**: 구글 시트는 자동으로 저장되니 그냥 끄시면 됩니다.
4.  **반영**: 챗봇을 **새로고침(F5)** 하면 바뀐 내용이 반영됩니다.

---

## 📊 3. 시장 상황 알려주기 (전문가 리포트)
"요즘 분위기 어때?"라는 고객 질문에 답하기 위해, 소장님의 지식을 입력하는 곳입니다.

1.  폴더 안에 있는 **`market_factors.md`** 파일을 엽니다. (메모장으로 열립니다)
2.  질문 항목(금리, 매수심리 등) 아래에 소장님의 생각을 자유롭게 적으세요.
    *   *예: "지금은 급매물만 소화되는 시기라..."*
3.  **저장(Ctrl+S)** 하고 닫으세요.
4.  챗봇을 **새로고침(F5)** 하면 AI가 이 내용을 공부합니다.

---

## 🖼️ 4. 이미지 비교 기능
고객에게 인테리어 전/후나, A타입/B타입 평면도를 비교해주고 싶을 때 씁니다.

1.  채팅창 오른쪽 위의 **[이미지 비교]** 버튼을 누릅니다.
2.  창이 뜨면 비교하고 싶은 설명(예: "작은방 확장 전후")을 하면서 슬라이더를 움직여 보여줍니다.
3.  (고급) 다른 사진을 넣고 싶으면 하단 입력창에 인터넷 주소를 넣고 [불러오기]를 누르세요.
