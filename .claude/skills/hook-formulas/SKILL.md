# Hook Formulas

첫 문장에서 독자의 주목을 끄는 훅 작성 공식 및 템플릿입니다.

## Hook의 중요성

- Instagram: 첫 줄 (더보기 전 노출)
- YouTube: 첫 10초
- Shorts/Reels: 첫 3초
- Email: 제목 + 첫 문장

## Hook Formula Categories

### 1. 질문형 (Question)

**공식**: "[타겟]도 [문제/고민] 있으신가요?"

**예시**:
- "아직도 인스턴트 커피 드세요?"
- "출근 전 카페 줄, 지치지 않으셨나요?"
- "홈카페 시작하고 싶은데 뭐부터 해야 할지 모르겠다고요?"

### 2. 반전형 (Contrast)

**공식**: "[일반적 믿음]? [반전 사실]"

**예시**:
- "비싼 장비 없이도 바리스타 커피 만들 수 있어요"
- "3분이면 카페 부럽지 않은 커피 완성"
- "인스턴트 가격으로 스페셜티 커피를"

### 3. 숫자형 (Number)

**공식**: "[숫자]가지 [혜택/방법] / [숫자]분 만에"

**예시**:
- "3분 만에 완성하는 프리미엄 커피"
- "5가지 드립백 추출 꿀팁"
- "1분에 끝내는 콜드브루 레시피"

### 4. 이익형 (Benefit)

**공식**: "[결과]를 원하신다면 [행동]"

**예시**:
- "집에서 카페 품질 커피를 원하신다면"
- "바쁜 아침에도 여유를 갖고 싶다면"
- "월 5만원으로 매일 스페셜티 커피를"

### 5. 공감형 (Empathy)

**공식**: "[상황] 때문에 [감정]이시죠? [해결책]"

**예시**:
- "아침마다 시간에 쫓기느라 커피도 제대로 못 마시죠?"
- "카페 가격이 부담스러우셨죠?"
- "집에서 만든 커피가 맛없었던 경험 있으시죠?"

### 6. 호기심형 (Curiosity)

**공식**: "[놀라운 사실/비밀]" / "왜 [대상]은 [행동]할까?"

**예시**:
- "바리스타들이 집에서 쓰는 커피 비밀"
- "이 드립백을 6개월째 재구매하는 이유"
- "아무도 알려주지 않는 홈카페 꿀팁"

### 7. 도전형 (Challenge)

**공식**: "당신도 [결과] 할 수 있습니다" / "[시간] 안에 [결과]"

**예시**:
- "초보도 3분이면 카페 커피 완성!"
- "일주일 만에 홈카페 마스터 되기"
- "이거 한 번 마시면 인스턴트 못 마셔요"

## Platform-Specific Hooks

### Instagram Feed
```
[짧고 강렬한 질문/진술]

예시:
"아직도 인스턴트요? ☕"
"바쁜 아침, 딱 3분만 투자하세요"
"집에서 이게 돼요? 네, 됩니다 ✨"
```

### Instagram Reels / Shorts
```
[1-3초 내 주목 / 시각+텍스트]

예시:
자막: "아직도 인스턴트?"
자막: "3분 만에 카페 커피"
자막: "이것만 알면 홈카페 끝"
```

### YouTube 제목
```
[키워드 + 호기심/이익]

예시:
"[홈카페] 3분 만에 완성하는 드립백 커피 | 초보자 가이드"
"바리스타가 알려주는 드립백 100% 활용법 (이것만 알면 됨)"
"월 3만원으로 매일 스페셜티 커피 마시는 방법"
```

### Email 제목
```
[개인화 + 호기심/이익]

예시:
"[이름]님, 아침 커피 3분 루틴 만들기"
"카페 안 가도 되는 이유 (첫 구매 15% 할인)"
"이번 주만 - 에티오피아 예가체프 한정 입고"
```

## A/B Testing Variations

같은 내용, 다른 훅:

**주제**: 드립백 신제품 출시

| 유형 | 훅 |
|------|-----|
| 질문형 | "카페 줄 서기 지치셨나요?" |
| 숫자형 | "3분 만에 카페 품질 커피" |
| 이익형 | "집에서 즐기는 스페셜티 커피" |
| 호기심형 | "바리스타들의 비밀 레시피" |
| 공감형 | "바쁜 아침, 커피는 포기 못하죠?" |

## Hook Checklist

- [ ] 첫 문장에서 주목을 끄는가?
- [ ] 타겟 페르소나와 관련 있는가?
- [ ] 읽고 싶은 호기심을 유발하는가?
- [ ] 약속한 내용을 본문에서 전달하는가?
- [ ] 채널 특성에 맞는가?
- [ ] 브랜드 톤앤매너에 맞는가?

## File Storage

카피 및 훅 문서는 프로젝트의 표준 디렉토리 구조에 저장합니다.

```
{project}/
├── reports/
│   └── content/          # 콘텐츠 문서 저장 위치
│       ├── hook-variations.md
│       └── copy-variations/
│           ├── instagram-hooks.md
│           └── youtube-titles.md
└── tmp/                  # 초안 및 A/B 테스트 파일
```

### 파일 명명 규칙

```
{project}/reports/content/{type}/{channel}-{name}-{date}.md

예시:
reports/content/copy-variations/instagram-hooks-20240115.md
reports/content/copy-variations/youtube-titles-20240115.md
```

이 스킬은 `conversion-copywriter` 및 `script-writer` 에이전트가 훅/헤드라인 작성 시 참조합니다.
