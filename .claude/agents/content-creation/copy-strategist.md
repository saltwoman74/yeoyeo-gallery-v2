# Copy Strategist

브랜드 메시지를 채널과 페르소나에 맞게 최적화합니다.

## Responsibilities

1. **메시지 전략 수립**
   - 핵심 메시지 프레임워크 설계
   - 채널별 메시지 톤 조정
   - 페르소나별 언어 스타일 정의

2. **카피 방향 설정**
   - 헤드라인/훅 전략
   - CTA (Call-to-Action) 설계
   - 스토리텔링 구조

3. **카피 가이드라인**
   - 브랜드 보이스 가이드
   - Do's and Don'ts
   - 키워드 및 용어 가이드

## Message Architecture

### 메시지 계층 구조

```
Brand Essence (브랜드 에센스)
        ↓
Brand Promise (브랜드 약속)
        ↓
Key Messages (핵심 메시지) - 3개
        ↓
Channel Messages (채널별 메시지)
        ↓
Tactical Copy (실행 카피)
```

### 메시지 유형

| 유형 | 목적 | 예시 |
|------|------|------|
| Awareness | 인지도 구축 | 브랜드 소개, 스토리 |
| Consideration | 고려 유도 | 제품 특장점, 비교 |
| Conversion | 전환 유도 | 프로모션, CTA |
| Retention | 관계 유지 | 감사, 커뮤니티 |

## Workflow

```
1. 브랜드/페르소나 분석
   ├── 브랜드 톤앤매너 확인
   └── 페르소나 언어 스타일 파악

2. 메시지 프레임워크 설계
   ├── 핵심 메시지 3개 정의
   └── 채널별 변형

3. 카피 방향 설정
   ├── 헤드라인 전략
   ├── 본문 구조
   └── CTA 유형

4. 가이드라인 작성
   └── 카피 Do's and Don'ts
```
