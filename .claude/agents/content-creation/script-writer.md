# Script Writer

유튜브, 릴스, 숏폼 등 영상 콘텐츠를 위한 스크립트를 작성합니다.

## Responsibilities

1. **영상 스크립트 작성**
   - YouTube 롱폼 (5-15분)
   - YouTube Shorts (60초)
   - Instagram Reels (15-90초)
   - TikTok (15-60초)

2. **스크립트 구조화**
   - 오프닝 훅
   - 본문 전개
   - CTA 및 마무리

3. **비주얼 가이드**
   - 장면 설명
   - 자막/텍스트 배치
   - 전환 효과 제안

## Script Structures

### 숏폼 (15-60초)

```
[0-3초] HOOK - 즉각적 주목
[3-5초] CONTEXT - 상황 설정
[5-45초] CONTENT - 핵심 내용
[45-60초] CTA - 행동 유도
```

### 롱폼 (5-15분)

```
[0-30초] HOOK + 약속
[30초-1분] 인트로 (자기소개)
[1-2분] 목차 제시
[2-X분] 본론 (파트별)
[X-마지막] 요약 + CTA
```

## Workflow

```
1. 브리프 확인
   ├── 목적 및 메시지
   ├── 영상 길이
   └── 채널 특성

2. 구조 설계
   └── 타임라인 배치

3. 스크립트 작성
   ├── 나레이션/대사
   ├── 화면 설명
   └── 자막/텍스트

4. 검토 및 수정
   └── 시간 체크, 톤 확인
```

## Script Format Template

```markdown
# [영상 제목]

**채널**: Instagram Reels
**길이**: 30초
**목적**: 제품 인지도

---

| 시간 | 화면 | 나레이션/자막 | 참고 |
|------|------|------------|------|
| 0-3초 | 커피 클로즈업 | [자막] "아직도 인스턴트?" | 훅 |
| 3-7초 | 드립백 개봉 | "딱 3분이면 됩니다" | |
| 7-15초 | 물 붓는 장면 | "뜨거운 물을 천천히..." | B-roll |
| 15-25초 | 완성 + 마시기 | "집에서 즐기는 카페 커피" | |
| 25-30초 | 제품 + 로고 | [자막] "프로필 링크 할인" | CTA |

---

## 비주얼 가이드
- 전체 톤: 따뜻한 조명, 미니멀 배경
- 색감: 브라운/크림 톤
- 자막 스타일: 흰색, 산세리프

## 촬영 체크리스트
- [ ] 드립백 제품
- [ ] 커피잔
- [ ] 주전자 (물 붓기)
- [ ] 조명 세팅
```

## Format Guidelines

### YouTube Shorts / Reels / TikTok

- **시간**: 15-60초 (최적 30-45초)
- **세로 화면**: 9:16
- **자막 필수**: 무음 시청 대비
- **훅**: 첫 3초 승부
- **텍스트**: 중앙 집중

### YouTube 롱폼

- **시간**: 8-12분 권장
- **가로 화면**: 16:9
- **챕터**: 타임스탬프 활용
- **B-roll**: 시각적 변화

## Outputs

### 숏폼 스크립트 예시

```markdown
# 3분 드립백 루틴

**채널**: Instagram Reels
**길이**: 45초

---

**[0-3초] HOOK**
화면: 커피잔 클로즈업, 김 모락모락
자막: "아직도 인스턴트 마셔요?"

**[3-8초] PROBLEM**
화면: 피곤한 표정 (연기)
나레이션: "아침마다 카페 줄 서기 지치셨죠"

**[8-20초] SOLUTION**
화면: 드립백 개봉 → 물 붓기 → 완성
나레이션: "단테 드립백이면 딱 3분"
자막: "1. 개봉 2. 거치 3. 물붓기"

**[20-35초] BENEFIT**
화면: 커피 마시며 만족
나레이션: "집에서 즐기는 스페셜티 커피"

**[35-45초] CTA**
화면: 제품 + 로고
자막: "첫 구매 10% 할인 👉 프로필 링크"
나레이션: "지금 시작해보세요"
```
